About RideEase

RideEase is a ride booking system developed as part of a student project. It demonstrates the implementation of:

  • MVC Architecture with Spring Boot
  • Multiple Design Patterns including Factory, Builder, Singleton, and Command
  • Database Integration with JPA and Hibernate
  • Command Line Interface & Web Interface

This project was developed by a team of 4 students.

System Features

User Management

Register, login, and manage user profiles

Driver Management

Register drivers, manage vehicles and availability

Ride Booking

Book rides, track status, and get fare estimates

Payments & Ratings

Process payments and rate completed rides
